# Portfólio - Matheus | Desenvolvedor Full Stack

Este repositório contém o código do meu portfólio profissional, desenvolvido para apresentar meus projetos, habilidades e experiências como desenvolvedor.

O objetivo do portfólio é demonstrar minhas competências em desenvolvimento web Full Stack, além de apresentar aplicações reais desenvolvidas para empresas e projetos pessoais.

##  Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* TypeScript
* React
* Node.js
* Python
* MySQL / PostgreSQL
* Git e GitHub

##  Projetos apresentados no portfólio

Alguns dos projetos exibidos no site:

* **NOC Dashboard**
  Painel de monitoramento de rede para visualização do status de equipamentos e alertas operacionais.

* **Intranet Acessanet**
  Sistema interno desenvolvido para centralizar ferramentas, sistemas e informações da empresa.

* **Área do Cliente – Projecta**
  Plataforma web segura para acesso de clientes a informações e serviços da empresa.

* **Hotspot Portal**
  Portal de autenticação Wi-Fi com página personalizada para acesso à rede.

* **Site Sibcare**
  Website institucional para clínica de estética.

* **Site Acessanet**
  Site institucional desenvolvido para provedora de internet.

* **Site Tico Carretos**
  Website institucional para empresa de transporte e carretos.

Este portfólio foi desenvolvido para apresentar meus projetos e evolução como desenvolvedor, além de servir como vitrine das tecnologias que utilizo no desenvolvimento de aplicações web.
